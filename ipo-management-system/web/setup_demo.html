Setup Instructions to Enable Full Functionality

1. Install PHP

Download PHP from https://windows.php.net/download/

Extract to C:\php and add to system PATH

php -v

2. Install MySQL/XAMPP

Option A: Download XAMPP from https://www.apachefriends.org/

Option B: Install MySQL separately

3. Setup Database

Create database named ipo_management_system

Run SQL files in this order:

database/schema/01_create_tables.sql
database/schema/02_constraints_triggers.sql
database/procedures/allotment_procedures.sql
database/views/reporting_views.sql
database/test_data/sample_data.sql

4. Start PHP Server

Open PowerShell in the web directory and run:

cd "c:\Users\HP\OneDrive\Documents\dbproject\ipo-management-system\web"
php -S localhost:8000

Or double-click start_server.bat

5. Access Full Application

Open browser and go to: http://localhost:8000
